## 1. 실험 의도

기존 12x12 실험(`21_1757_12x12_signaltrap`)의 **드라이버 운동(가·감속) 모델을
현실화**하고, **재학습 없이** 같은 학습 모델로 재실험해 그 영향을 측정한다.

**변경 내용 — 노드 통과 시 감속 규칙**

| 구분 | 기존 | 개정 |
|---|---|---|
| 좌·우회전 | 회전 속도로 감속 | **유지** — 회전 속도(좌 30·우 20 km/h)로 진입 |
| 직진 | 다음 노드가 신호이면 일률 감속(20~30 km/h) + 모든 링크 진입 시 0.7배 감속 | **개선** — 노드에서의 명시적 감속 없음. 직전 링크 순항속도를 이어받아 **링크 간 속도 차이만큼만** 가·감속 |

기존 모델은 직진 통과 차량도 신호 교차로마다 감속했다(비현실적). 개정 모델은
직진 시 노드 감속을 없애고, 회전 시에만 감속을 유지한다. 회전 감속은 회전해
들어가는 링크의 진입속도(`v_entry`)로 반영된다.

`environment.py` 의 실제 동역학과 `StaticFuelDijkstra` 의 비용 추정 모델에
동일하게 적용해 플래너-환경 정합을 유지했다.

## 2. 실험 설계

- 토폴로지·신호·속도·경로·반복: `21_1757` 과 동일 (144 노드, diag_main n1→n144,
  peak 08:00, 30회). 환경 상세는 `21_1757_12x12_signaltrap/report.md` §2 참조.
- 모델: ① shortest_dijkstra ② static_fuel_dijkstra ③ rl_base ④ rl_signal
  ⑤ rl_signal_attention — **기존 학습 가중치 그대로 사용(재학습 없음)**.
- 즉 RL 3종은 *옛 드라이버 모델로 학습*된 정책을 *새 드라이버 모델 환경*에서
  평가 — 동역학 변화에 대한 일반화도 함께 관찰된다.

## 4. 핵심 발견

1. **신호 회피 효과는 그대로, 오히려 강화됨.** 최단거리 모델(대각 코리도)이
   여전히 최악(1212 mL)이고, 신호·연료 인식 4개 모델은 모두 우회로로
   **15~21% 절감**(기존 12~20%). 사전검증상 peak 코리도-우회로 격차도
   25.5%→29.2% 로 커졌다. 드라이버 모델 개정이 핵심 결론을 흔들지 않는다.

2. **전체 연료 하락 — 직진 노드 감속 제거 효과.** 모든 모델의 연료가 감소했다.
   직진 차량이 신호 교차로마다 감속·재가속하던 군더더기 연료가 사라졌기 때문.
   특히 **고속 간선을 직진 주행하는 우회로**가 이득을 크게 봐(static_fuel
   −33), 함정 코리도(저속 직진, −20) 대비 우회로의 상대 우위가 더 커졌다.

3. **RL 내부 순위 재편 — 미재학습의 영향.** 기존엔 ⑤ attention < ④ signal
   < ③ base 였으나, 재실험에서는 **③ rl_base(981) 가 RL 중 최저**가 됐다
   (rl_base −102 로 가장 큰 폭 개선). RL 3종은 *옛 드라이버 모델*로 학습된
   정책이라, 동역학이 바뀐 환경에서의 적합도가 모델마다 다르게 나타난 결과다.
   rl_base 의 정책이 새 동역학에 우연히 더 잘 맞았고, 신호를 더 정교히 보는
   signal/attention 은 학습 시 가정한 감속 패턴이 사라지며 상대적으로 손해를
   봤다. **이는 정책 성능 자체의 우열이 아니라 "학습 환경 ≠ 평가 환경"
   불일치의 산물**이다.

4. **static_fuel_dijkstra 가 여전히 최저(958 mL, −21%).** 환경 모델을 가진
   고전 시간의존 Dijkstra 가 상한선. 비용 모델을 새 드라이버 규칙에 맞춰
   갱신했으므로 플래너-환경 정합은 유지된다.

## 5. 결론

드라이버 운동 모델을 현실화(직진 노드 감속 제거, 회전 감속 유지)해도 12x12
신호 회피 실험의 핵심 결론 — **신호·연료 인식 경로탐색이 최단거리 대비
연료 15~21% 절감** — 은 견고하게 유지되며 효과 크기는 오히려 증가했다.
다만 RL 3종은 옛 동역학으로 학습돼 있어, 동역학이 바뀐 환경에서 내부 순위가
재편됐다. 정확한 모델 간 비교를 위해서는 **새 드라이버 모델로 재학습이
필요**하다(본 실험은 사용자 지정에 따라 재학습 없이 수행).

## 6. 한계 및 문제점

1. **학습-평가 환경 불일치** — RL 3종은 옛 드라이버 모델로 학습된 가중치를
   그대로 사용했다. §4-3 의 순위 재편은 이 불일치의 직접적 결과로, 새
   드라이버 모델 기준의 공정한 RL 비교는 재학습 후에 가능하다.
2. **회전 감속의 단순화** — 회전 감속을 회전 후 링크의 진입속도로 모델링한다.
   회전 직전의 감속 구간(직전 링크 말단)은 명시적으로 적분하지 않는다(감속
   연료는 작아 영향 경미, 재가속 연료는 정상 반영).
3. **신호 회피 이득의 첨두 의존성** — off_peak 에서는 최단 코리도가 유리
   (사전검증). 본 실험은 peak 한정.
4. **도달률 97%** — 5개 모델 중 RL 3종이 30회 중 1회씩 미도달(옛 동역학 학습
   정책의 새 환경 적응 한계). 연료 평균이 소폭 부풀려졌을 수 있다.
